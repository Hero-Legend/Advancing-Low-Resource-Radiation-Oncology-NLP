# Results Directory

This directory is kept in the public release so the repository layout remains compatible with the released scripts.

## Intended Use

Use this folder for regenerated outputs such as:

- run summaries
- robustness exports
- analysis intermediates
- locally reproduced tables and plots

## Why It Is Mostly Empty

We intentionally do not ship:

- manuscript-facing summary tables
- remote training checkpoints
- transient server-side artifacts

The goal of this repository is to release a clean reproducibility package, not a paper-packaging dump.
